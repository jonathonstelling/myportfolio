Active Form

Active Form

Design Team: Jonathon Stelling, ..

In this project we had to design an active form. Simply put an active form is a tangible user interface (TUI) that couples action and reaction in a single modality. This coupling is achieved by eliminating the distinction between input and output. The active form is physically deformed by the user and change both shape and appeanrance and similarly it is deformed as output feedback. The deformation, thanks to the coupling is on the same aspect of the device (i.e. change of shape, temperature, colour...).

Concepts

We first looked into nature, where we would be able to create an active form

Then we came up with some initial ideas, based around a tree/plant and some of the different ideas of how the tree would change and move as the user interacted with it. The tree/plant will mainly work as an alarm to wake the user up

Final Design

Our final design is a combination of some of our initial ideas. The final design features a tree/plant on top of a see through plant pot and a seed suspended inside of the plant pot. The user will use their hand to set the height of the tree which will indicate how long they wish to sleep for and set the alarm. Once the alarm has been set the tree will then return to its flat position on the plant pot and gradually rise through the night until it reaches the height the user set it to, then the alarm will sound and the radio will play. The user will also be able to change the room temperature in the room by either breathing on the tree to increase the room temperature, or blow on the tree to decrease the room temperature.

Prototype

With the use of a vacuum former, laser cutter and other manfuacturing methods we were able to create a working prototype. We decided to use a telescopic car aerial for moving the tree up and down as it was the most cost effective way of doing so, and using an ultrasonic sensor to control the height of the tree.
