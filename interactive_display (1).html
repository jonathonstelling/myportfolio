Interactive Display

In this project I created my own interactive display for my car which included a variety of systems such as a Playstation 2 and an Android PC.

-Android PC install and set-up

I wanted to design my own interactive system in my car, to do this I chose the Android platform as applications were readily avaliable. I first started to design the layout of the Android PC, by making it easy to choose regually used apps and also having key information displayed such as the weather, time and date. Almost every app that is avaliable on Google Play for the android market can be used within the car. Apps such as Netflix and Sky Sports which require WiFi to work can be used when parked up near to a WiFi connection, but also on the go by using a wireless hotspot which is common on most smartphones. It also supports USB storage devices so content such as movies, music and pictures can all be stored and displayed.

Playstation install and set-up:

Playstation 2 fitted under passenger seat, and is able to be slided backwards and forwards to be able to insert new discs etc...

By using wireless controllers this helped to remove the need for any wires to be visible in the cabin, thus giving a smarter finish.

User interface:

I fitted a proximity sensor underneath the dashboard, located infront of the screen. Whenever I wanted to change the input onto the screen, say either the to display the Android PC or Playstation 2 I simply had to swipe my hand over the proximity sensor and it would change to the next display. Using a proximity sensor under the dash made the final design smart and sleek.

Audio Selection:

One problem I faced while feeding 3 devices into the same screen was that there was no way to switch the audio channels. As I wanted to keep the standard AUX connection I had to split the audio connection. I then installed an audio switch box so I was then able to select my audio feed of my choice. The audio switch box was placed inside of the glovebox, making it easily accessible and also hidden.

Future developments:

Future developments to the interactive display are to continue developing version 2. In this version I have tried to incorporate the individuality of the vehicle by including the private licence plate animated for the start up screen.
