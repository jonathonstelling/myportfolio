Portfolio Game

I used Processing (Java) to create my portfolio game. I again chose Processing because it allowed me to design the game to be displayed on my website and android. The game works be either using the keyboard or mouse (Touchscreen) to move the character and discover more of my portfolio pages. Useful information is also displayed on the bottom of the screen, such as my name, website url, progress bar and a character speed selector.

Future developments for the game are to include weapons, and objects for the character to shoot so that the player can progress in the game.
